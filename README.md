> ###SimpleVideo 特性：
>> 
1. 面向接口, 易于对音视频解析API站点的扩展.
2. 为便于使用SimpleVideo, 提供hrome *sv*插件; 插件无需干预可自动更新.
3. 数据缓存: 自动缓存; 缓存手动刷新; 缓存清理.
4. 支持在线观看.
5. 无源站点广告，链接对外暴露，可自由下载.
6. 支持url rewrite, 访问更加直观
7. *容我想一会儿* ^_^
